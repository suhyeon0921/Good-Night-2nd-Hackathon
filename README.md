## env

```
NEXT_PUBLIC_BASE_URL='http://localhost:8080'
```

## 페이지

```
- / : 영화 목록
- /add : 영화 등록
- /movie/[id] : 영화 상세 정보, 리뷰
- /movie/edit/[id] : 영화 수정
```
